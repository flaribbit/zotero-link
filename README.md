# zotero-link

突然发现在 zotero 里复制笔记的时候，会以 markdown 格式复制，并且带着指向笔记的链接。

但是我正在使用的 notion 不支持这种链接，which is very bad。

于是就有了这个项目。

这个程序会监控剪贴板，当复制 zotero 笔记时替换为一个中转链接，当你粘贴到 notion 里面时，能够被正确识别为链接，点击可以跳转到 pdf 对应位置。
